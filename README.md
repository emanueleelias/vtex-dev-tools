# VTEX Dev Tools

Una suite centralizada de herramientas open-source diseñadas para desarrolladores de **VTEX IO**. Este repositorio contiene la [_landing page_ oficial](https://emanueleelias.github.io/vtex-dev-tools/) que actúa como portal para todo el ecosistema.

## 🛠 Herramientas incluidas

### 1. [VTEX Landing Generator](https://github.com/emanueleelias/vtex-landing-generator)
Generador visual drag & drop para construir estructuras de bloques VTEX IO y exportarlas como JSONC. Evita escribir código a mano cuando armas tus layouts en el `store-theme`.

### 2. [VTEX CSS Sanitizer](https://github.com/emanueleelias/vtex-css-sanitizer)
Limpia y optimiza las hojas de estilo encontrando clases CSS huérfanas y `blockClass` sin uso real en tu proyecto. Disponible como CLI y aplicación GUI multiplataforma.

### 3. [VTEX Deploy Helper](https://github.com/emanueleelias/vtex-deploy-helper)
Automatiza los procesos repetitivos de despliegue en VTEX IO. Desde publicación de nuevas custom apps hasta actualización de releases maje y patch, facilitando la integración con `site-editor`.

## 🚀 Sobre esta Landing Page

La página principal del proyecto está construida como un sitio estático moderno con:
- Arquitectura HTML/CSS/JS sin dependencias (ideal para GitHub Pages).
- Paleta de diseño _Dark Mode_ con _glassmorphism_.
- Soporte de Internacionalización (i18n) interactivo (Español / Inglés).
- Sistema responsivo y animaciones al hacer scroll (scroll reveal).
- Funcionalidades utilitarias como "Copia al portapapeles" y fetch automático de versiones actualizadas de NPM para los CLIs.

### Ejecución Local

Para probar o modificar esta web de manera local, simplemente abre `index.html` en tu navegador, o utiliza cualquier servidor local servido desde la raíz (como Live Server en VS Code).

```bash
npx serve .
```
